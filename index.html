Package playwright

Python package playwright is a Python library to automate Chromium, Firefox and WebKit with a single API. Playwright is built to enable cross-browser web automation that is ever-green, capable, reliable and fast. For more information you'll find the documentation for the sync API here and for the async API here.

class FloatRect(ApiType):

    y: float
    width: float

        if not dict:

        return FloatRect(dict["x"], dict["y"], dict["width"], dict["height"])

    def __init__(self, x: float, y: float, width: float, height: float):
        self.x = x
        self.y = y
        self.width = width
